# Design Documentation

C4 architecture diagrams and system documentation. Use Mermaid syntax (`.mmd` files).

## Structure

- **Architecture/** – Deployment and infrastructure
- **Components/** – C4 component diagrams
- **Containers/** – C4 container diagrams
- **Context/** – C4 context diagrams
- **Data/** – Entity relationships, data models
- **Flows/** – Sequence diagrams, user workflows
- **States/** – State diagrams

See `.cursor/rules/design-documentation-standards.mdc` for diagram types and naming conventions.
